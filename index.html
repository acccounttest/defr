defr

is an — at present: proof-of-concept — simple usage pattern and lightweight JavaScript library for deferred loading and localStorage caching of external CSS and JavaScript resources.

Documentation

An up-to-date documentation including a compatibility chart is available at the GitHub repository.

Examples

There are four demo pages that you can use to test defr. Please choose the library version that complies to your browser:

  • Simple asset loading in modern browsers (modern-simple)
  • Simple asset loading in older browsers (polyfilled-simple)
  • localStorage caching in modern browsers (modern-localstorage)
  • localStorage caching in older browsers (polyfilled-localstorage)
